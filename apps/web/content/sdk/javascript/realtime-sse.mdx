## Conexão automática

O SDK abre automaticamente uma conexão SSE durante `init()`. Você não precisa configurar nada — as flags são atualizadas em background sem polling.

```ts
const client = new CanaryGate('cg_key')
await client.init() // abre SSE aqui

// A partir deste momento, client.getFlag() sempre retorna
// o valor mais recente sem precisar de refetch manual
```

## Reagindo a mudanças (React)

Para que componentes React reflitam mudanças de flags em tempo real, você precisa de um mecanismo de reatividade. O SDK não inclui bindings React nativos, mas é simples de implementar:

```tsx
import { CanaryGate } from '@canarygate/js'
import { useState, useEffect, useRef } from 'react'

function useFlagEnabled(client: CanaryGate, key: string): boolean {
  const [enabled, setEnabled] = useState(
    () => client.getFlag(key)?.enabled ?? false
  )

  useEffect(() => {
    const interval = setInterval(() => {
      const current = client.getFlag(key)?.enabled ?? false
      setEnabled(current)
    }, 500)

    return () => clearInterval(interval)
  }, [client, key])

  return enabled
}
```

<Callout type="info">
  Uma abordagem mais eficiente é usar um event emitter. O SDK emite eventos
  internamente — consulte a documentação avançada para escutar mudanças
  diretamente via callback.
</Callout>

## `isStale()`

Retorna `true` se o SDK não recebeu nenhuma atualização SSE dentro do `staleThreshold` (padrão: 60 segundos).

```ts
if (client.isStale()) {
  // Dados podem estar desatualizados
  // Considere recarregar ou mostrar aviso ao usuário
}
```

Use `isStale()` para monitorar a saúde da conexão em dashboards críticos.

## `getLastSyncAt()`

Retorna a `Date` da última sincronização bem-sucedida, ou `null` se `init()` ainda não completou.

```ts
const lastSync = client.getLastSyncAt()
if (lastSync) {
  console.log('Flags atualizadas há:', Date.now() - lastSync.getTime(), 'ms')
}
```

## `disconnect()`

Fecha a conexão SSE e libera os recursos. Sempre chame ao desmontar:

```ts
// React:
useEffect(() => {
  return () => client.disconnect()
}, [])

// Node.js / script:
process.on('SIGTERM', () => client.disconnect())
```

Após `disconnect()`, `getFlag()` continua retornando os últimos valores conhecidos, mas não receberá mais atualizações.

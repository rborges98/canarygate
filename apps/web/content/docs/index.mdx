## O que é o CanaryGate?

CanaryGate é uma plataforma de **feature flags** para times que querem controle sobre rollout sem a complexidade e o custo de soluções enterprise.

Com o CanaryGate você:

- Ativa ou desativa features instantaneamente, **sem redesploiar**
- Faz rollout gradual para uma porcentagem crescente de usuários
- Agenda ativações automáticas com horário definido
- Recebe mudanças em **tempo real** via SSE — o SDK atualiza sem precisar de polling
- Organiza flags por projeto e environment, com histórico completo de auditoria

## Como funciona, em 3 passos

1. **Crie uma flag** no painel web — dê um nome, escolha o tipo (boolean ou rollout) e selecione o environment.
2. **Integre o SDK** na sua aplicação com `npm install @canarygate/js`.
3. **Controle o rollout** pelo painel — mudanças chegam para os clientes em tempo real via SSE.
